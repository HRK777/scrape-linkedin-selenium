💡 #SwiftTip: Using Result<T, Never> to represent impossible code flow

Set Result.Failure type to Never in case error is impossible or can be replaced with a value. E.g., pass an empty array if an array of elements has failed to load.

The compiler is smart enough and does not require us to handle the failure case in the switch statement.
